# 💎 Jewelry Product Analysis System

A complete **Machine Learning and NLP-based system** to analyze, predict, and recommend jewellery products.  
This project demonstrates the use of **data science, deep learning, and natural language processing (NLP)** techniques on jewelry datasets.

---

## 📘 Overview

The **Jewelry Product Analysis System** performs:

1. **Data Creation / Loading**
   - Generates a mock jewelry dataset (`data/jewelry_dataset.csv`).
2. **Price Prediction (Regression)**
   - Predicts jewelry prices using **Linear Regression** and **TensorFlow**.
3. **Product Categorization (Classification)**
   - Classifies jewelry type (Necklace, Ring, Bracelet, etc.) using **Random Forest**.
4. **Product Recommendation System**
   - Suggests similar products using **Content-Based Filtering** (`NearestNeighbors`).
5. **Customer Sentiment Analysis**
   - Analyzes product reviews using **VADER Sentiment Analyzer** from NLTK.
6. **Deep Learning Integration**
   - Trains a neural network in **TensorFlow** for price prediction.
